## Local testing

To run the Stencil locally, with backend and local database, run these commands

/the-stencil-parent
`mvn clean install`

/the-stencil-parent/dev-tools/demo-app
`mvn compile quarkus:dev`

/the-stencil-composer
`yarn start`
